# Spring Boot Todo App

Spring Boot と Thymeleaf を利用したTodoアプリです。

## 機能

- タスク追加
- タスク編集
- タスク削除
- 完了/未完了切替
- 優先度設定（高・中・低）
- 優先度色分け表示
- キーワード検索
- 登録件数表示
- 完了件数表示

## 使用技術

- Java 21
- Spring Boot 4
- Thymeleaf
- Bootstrap 5
- Maven

## 画面

Todo一覧画面
