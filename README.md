# TODO App

Spring Boot と Thymeleaf を使用して作成した TODO アプリです。

## 概要

タスクの追加・完了・未完了への戻し・削除ができます。

## 使用技術

* Java
* Spring Boot
* Thymeleaf
* Bootstrap 5
* Git
* GitHub

## 実装機能

* タスク追加
* タスク完了
* タスク未完了へ戻す
* タスク削除
* 完了 / 未完了の一覧表示
* BootstrapによるUI改善
